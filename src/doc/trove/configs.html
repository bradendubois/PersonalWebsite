configs

A collection of shell / bash scripts used to install dotfiles for customizing a terminal / shell, and installing software.

I made this to facilitate setting up a new Arch-based distro for everyday use and development.

Notable installations:

  • oh-my-bash
  • Tweaked .bash_profile and .bashrc configs

Allows backing up of old config files.

Learned a bit of shell scripting in a second-year CS class, so this was a lot of fun to put it into practice.
